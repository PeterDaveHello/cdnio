# CDN.io 

CDN.io is there to always give you get the latest version of hosted JavaScript libraries on public CDNs.

## Installation

Add this line to your application's Gemfile:

    gem 'cdnio'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cdnio
